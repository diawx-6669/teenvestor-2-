# 📈 Teenvestor — Financial Literacy Ecosystem for Youth

**Teenvestor** — это передовая образовательная платформа, трансформирующая процесс изучения финансовой грамотности через геймификацию и интерактивное обучение. Проект ориентирован на подростков и построен на концепции "финансовых островов", обеспечивая глубокое вовлечение в тему управления капиталом[cite: 1].

---

## 🏗 Технологическая архитектура (Tech Stack)

Проект спроектирован с учетом современных стандартов веб-разработки:

* **Frontend Engine:** Реализован на **React.js**, что обеспечивает реактивность интерфейса и бесшовный переход между образовательными модулями[cite: 1].
* **Deployment Pipeline:** Платформа полностью развернута через **GitHub Pages**, что гарантирует высокую доступность и стабильность работы[cite: 1].
* **Internationalization (i18n):** Внедрена система многоязычности, позволяющая переключаться между **Русским (RU)**, **Казахским (KZ)** и **Английским (EN)** языками «на лету»[cite: 1].

---

## 🧠 Геймифицированные модули (Key Features)

Teenvestor использует психологические триггеры вовлечения для удержания внимания пользователя:

1. **Learning Path (Образовательные острова):**
    * Каждый "остров" представляет собой тематический кластер знаний (например, "Базовые накопления", "Рынок акций", "Личные финансы")[cite: 1].
    * Пошаговая структура обучения предотвращает информационную перегрузку и закрепляет навыки[cite: 1].

2. **Economic Simulation (Экономическая модель):**
    * Система использует виртуальную валюту для поощрения активности[cite: 1].
    * Интеграция игровых механик (рулетка, наградные кейсы) превращает теоретические задания в захватывающий процесс[cite: 1].

3. **Competitive Social Layer:**
    * Таблицы лидеров стимулируют здоровое соревнование между пользователями[cite: 1].
    * Личные профили позволяют отслеживать прогресс обучения и накопленный виртуальный капитал[cite: 1].
